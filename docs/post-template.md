# Post template

Use este modelo para novos artigos em `_posts/AAAA-MM-DD-titulo.md`.

```yaml
---
layout: post
title: "Titulo do artigo"
date: 2026-04-28 10:45:00 -0300
description: Resumo curto do artigo para cards, SEO e listagens.
featured: true
reading_time: 6 min
tags:
  - jekyll
  - markdown
  - yaml
cover_image: /assets/images/hero.jpg
cover_alt: Descricao curta da imagem de capa
---
```

Depois do `front matter`, o restante do arquivo e Markdown puro.
